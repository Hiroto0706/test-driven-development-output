# Requirements this practice

## WyCash 多国通貨システム要件

| 要件 | 説明 |
|------|------|
| 1. 多国通貨対応 | 価格と合計に通貨情報を追加する（USD、CHF など） |
| 2. 通貨換算 | 異なる通貨間の為替レートを定義する |
| 3. 基準通貨計算 | すべての合計を基準通貨（USD など）に換算する |
